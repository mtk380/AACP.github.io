# 16. Roadmap

## Milestone Plan

### M1 — Foundation (2026 Q2)

- CometBFT integration
- ABCI 2.0 skeleton
- IAVL state storage
- Ed25519 + Protobuf baseline
- P2P connectivity and CI/CD

### M2 — Testnet Alpha (2026 Q3)

- 4-node testnet
- AMX baseline matching
- AAP core state machine
- Cap-UTXO basic flows
- Single-currency fiat escrow
- CLI + base SDK

### M3 — Testnet Beta (2026 Q4)

- Public testnet
- Full AMX matching
- WEAVE DAG engine
- Reputation v1
- Arbitration phase 1+2
- Dual-currency settlement (CNY + USD)
- T0–T3 deployment and bug bounty

### M4 — Security Audit (2027 Q1)

- Third-party audit
- TLA+ verification
- Red-team penetration testing
- Performance stress and tuning
- Anti-fraud v1
- Documentation/API consolidation

### M5 — Mainnet Launch (2027 Q2)

- Genesis and validator set bootstrapping
- Fiat gateway integration
- Governance module activation
- T4/T5 edge-node onboarding
- Initial insurance capitalization

### M6 — Ecosystem Expansion (2027 Q3+)

- Multi-language Agent SDK
- Developer marketplace growth
- Enterprise SLA templates
- Cross-chain interoperability
- Ecosystem funding and vertical solutions
